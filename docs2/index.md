# Lokstra (Docs v2 Draft)

**Modern Go Web Framework with two tracks: Router & Application Framework.**

- **Track 1 – Router**: use Lokstra like **Gin / Echo / Chi**.  
- **Track 2 – Application Framework**: use Lokstra like **NestJS / Spring Boot**, but idiomatic Go.

Pick your path:

- **New to Lokstra?** Start with [00 – Introduction](./00-introduction/).  
- **Want a better router?** See [01 – Router Guide](./01-router-guide/).  
- **Need full framework & DI?** See [02 – Framework Guide](./02-framework-guide/).  
- **Looking for signatures & config keys?** See [03 – API Reference](./03-api-reference/).
