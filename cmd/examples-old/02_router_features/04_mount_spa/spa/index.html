Single Page Application with Client-Side Routing

Current Route: /
Fallback File: ./spa/index.html

Welcome to Lokstra SPA!

This is a Single Page Application served by Lokstra using MountSPA.

How it works:

  • All unmatched routes fallback to this index.html file
  • Client-side JavaScript handles routing
  • API endpoints (like /api/*) are handled server-side
  • SPA routes (like /about, /dashboard) are handled client-side
