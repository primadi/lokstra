Lokstra Static Files Example

This HTML file is served from the public directory!

This page demonstrates how Lokstra can serve static HTML files along with CSS assets.

Available Static Resources:

  • /static/sample.txt - Text file from static directory
  • /assets/style.css - CSS file from assets directory
  • /public/index.html - This HTML file from public directory

API Endpoints:

  • /api/info - Server configuration information
  • /health - Health check endpoint
  • / - Root endpoint with available routes

Access this file at: http://localhost:8080/public/index.html
